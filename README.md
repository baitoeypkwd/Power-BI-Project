**Project Description**
This project presents a **Retail Sales Analytics Dashboard** built using **Power BI** to analyze business performance and customer behavior.
The dashboard provides insights into key metrics such as **total sales, profit, and orders**, along with sales trends over time.
It includes multiple analytical views such as **Executive Overview, Customer Insights, Product Performance, Store Performance, and Budget vs Actual Analysis**.
These dashboards help identify top-performing products, customer segments, and regional store performance.
Interactive visualizations such as **trend charts, ranking tables, maps, and variance analysis** allow users to easily explore the data and monitor business performance.
The goal of this project is to demonstrate how **data visualization and business intelligence tools** can support better decision-making.

**Tools:** Power BI, DAX, Data Visualization, Data Modeling
